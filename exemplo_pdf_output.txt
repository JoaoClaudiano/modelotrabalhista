ELEMENTOS VISUAIS NO PDF:

1. CABEÇALHO DA EMPRESA:
   - Nome da empresa (negrito, centralizado)
   - Endereço (negrito, centralizado)
   - Espaçamento: 6mm após cabeçalho

2. TÍTULO DO DOCUMENTO:
   - Linha decorativa superior (0.4pt de espessura, largura total)
   - Espaço de 3mm
   - PEDIDO DE DEMISSÃO (centralizado, fonte 12pt, negrito)
   - Espaço de 3mm
   - Linha decorativa inferior (0.4pt de espessura, largura total)
   - Espaço de 6mm após título

3. CORPO DO TEXTO:
   - Parágrafos justificados
   - Listas com bullet points (•)
   - Espaçamento entre parágrafos: 2.5mm
   - Fonte: 11pt

4. ASSINATURA:
   - Espaço extra antes: 5mm (2.5mm * 2)
   - Linha de assinatura: 1px, 280px de largura, centralizada
   - Label: "Assinatura do Funcionário" (centralizado, fonte 10pt)
   - Espaço: 4.5mm após assinatura

5. LINHA SEPARADORA (AJUSTADA):
   - Espaço antes: 2.5mm
   - Linha grossa: 0.4pt, largura total da página
   - Espaço depois: 5mm (AJUSTADO de 2.5mm para 5mm)
   - ↑ Este espaçamento agora corresponde ao HTML (~18px ≈ 4.76mm)

6. SEÇÃO "RECEBIDO POR":
   - Campos para preenchimento manual
   - Espaçamento adequado entre campos

================================================================================

DIFERENÇAS ENTRE HTML E PDF:

HTML (Pré-visualização):
- Usa CSS com pixels (px)
- Renderizado pelo navegador
- Separador tem margin: 18px 0

PDF (Download):
- Usa pontos e milímetros
- Gerado por jsPDF
- Separador tem espaçamento de 5mm após (ajustado)

RESULTADO: Agora o espaçamento é visualmente consistente entre HTML e PDF!
